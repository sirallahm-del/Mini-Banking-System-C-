# Mini-Banking-System-C-
A simple console program to manage bank accounts with file storage.
Mini Banking System in C

A simple console program to manage bank accounts with file storage.

Features

Create accounts (ID, Name, Balance)

Show all accounts

Deposit money into an account

Withdraw money from an account (with balance check)

Data is saved automatically in accounts.txt and loaded on next run

Skills learned

Structures (struct)

Arrays of structs

Loops and conditions

Functions

File handling (reading and writing)

Basic logic for real-world applications (deposit/withdraw)
